Installation: 
-Ganzen Ordner nach C:\Programme oder wo immer 
gewünscht kopieren. 

-Ordner Umbennen in FluFigSim oder wie gewünscht.

-Shortcut von FluFigSim11alpha auf Desktop erstellen. 

Progrmm kann auch direkt von der CD aus starten: 
Doppelklick auf FluFigSim11alpha.exe


Rechner-Leistung: 
Je nach Rechner-Leistung wird die Animation
langsam. Die erforderliche Rechner-Leistung kann
mit der Reduktion der Bildrate reduziert werden.
